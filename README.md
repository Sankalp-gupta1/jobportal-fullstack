🚀 JOB PORTAL FULL STACK PROJECT
📌 PROJECT DESCRIPTION

This is a full stack job portal application built using Node.js, Express, and MongoDB. It supports user registration, login, and authentication.

🎯 KEY FEATURES
User Registration
User Login
JWT Authentication
Password Encryption
REST API
🛠️ TECH STACK
Node.js
Express.js
MongoDB
Mongoose
JWT
Bcrypt.js
📁 PROJECT STRUCTURE

server/

models/
routes/
index.js
⚙️ HOW TO RUN
1. Install Dependencies

npm install

2. Start Server

node index.js

🔐 ENV VARIABLES

MONGO_URL=your_mongodb_url
JWT_SECRET=jobportal123
PORT=5000
